# Scripture Sources

This file tracks shared scripture-source rules and provenance notes used by both Wei-managed and Codex-managed projects.

## Shared Rule

Shorts should normally originate from scripture text represented by an existing channel long video, unless a project note explicitly authorizes a wider source scope.

When a Short uses scripture text:

* Record the scripture name.
* Record the source range or index, such as `第N句`, `第N段`, or `第N偈`.
* Keep selected Short text unique within the same scripture series unless Wei explicitly approves reuse.
* Record Chinese text, pinyin, English guide text, and any Japanese or Korean guide/search layer before rendering.
* Complete human text review for Wei-managed projects before render or upload.
* Complete provenance and safety review for Codex-managed experiments before upload.
