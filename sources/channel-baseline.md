# Channel Baseline Sources

This file records shared channel and published-output references used for comparison between Wei-managed and Codex-managed projects.

## Purpose

Use this file when a project needs common baseline context, such as:

* Which videos and Shorts existed before the Codex-managed experiment.
* Which published long video a Short should link back to.
* Which outputs are Wei-managed comparison data.
* Which outputs belong to the Codex-managed experiment.
