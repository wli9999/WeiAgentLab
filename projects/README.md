# Projects

This folder contains one manager-level note per project, grouped by ownership type.

Use `projects.md` as the registry of all projects. Use files in this folder for richer project context: purpose, ownership type, path, current goal, key files, open tasks, risks, and recent notes.

Project source files should usually remain in their own project folders outside Wei Agent Lab.

Shared source facts that may be used by more than one ownership type should live in `sources/`, not in an ownership-specific project note.

## Structure

```text
projects/
  README.md
  wei-managed/
    video-create.md
  codex-managed/
    codex-challange-shorts.md
```

Ownership types:

* `Wei-managed` means Wei controls the project direction and Codex assists.
* `Codex-managed` means Codex controls the experiment within recorded rules so it can be compared against Wei-managed work.
