# Templates

Reusable templates for Wei Agent Lab planning, execution, review, and documentation.

Copy a template into the relevant folder when starting new work, then replace placeholder text with project-specific details.

## Available Templates

* `project-note-template.md`
* `task-template.md`
* `research-summary-template.md`
* `test-report-template.md`
* `review-template.md`
* `decision-record-template.md`
* `video-create-new-video-template.md`
