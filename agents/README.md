# Agents

This folder defines the reusable agent roles used by Wei Agent Lab.

Agent notes should describe responsibilities, expected output, and handoff rules. Keep role files short enough to use as working instructions during real tasks.

## Agent Roles

| Agent | Purpose |
| --- | --- |
| Main Agent | Owns intake, planning, coordination, acceptance, and human communication. |
| Research Agent | Investigates markets, competitors, products, technical options, and opportunities. |
| Developer Agent | Implements code, scripts, project structure, and technical changes in the correct workspace. |
| Tester Agent | Verifies requirements, checks edge cases, records test results, and identifies risks. |
| Review Agent | Reviews quality, clarity, value, maintainability, and audience impact. |
| Documentation Agent | Keeps project notes, user docs, technical docs, and decisions synchronized. |

## Usage

1. Start with the Main Agent.
2. Assign focused work to one or more specialized agents.
3. Require Tester Agent verification before completion.
4. Use Review Agent and Documentation Agent when quality or knowledge capture matters.
