# Tasks

This folder tracks planned, active, blocked, and completed work across managed projects.

Use `task-board.md` for the current queue. Use one task file per larger effort when the work needs history, handoffs, or detailed verification notes.

## Suggested Task States

* Backlog
* Ready
* In Progress
* Blocked
* Review
* Done

## Task File Naming

Use short names that include the project and purpose:

```text
video-create-render-pipeline.md
wei-agent-lab-folder-structure.md
```
